# React 12-Week Odyssey

A comprehensive 12-week learning plan to master React development, from fundamentals to advanced concepts.

## Overview

This learning plan is designed to take you from React beginner to confident developer over 12 weeks. Each week builds upon the previous, with hands-on projects and practical exercises.

### Prerequisites

- Basic understanding of HTML, CSS, and JavaScript
- Familiarity with ES6+ features (arrow functions, destructuring, modules)
- Node.js and npm installed on your machine
- A code editor (VS Code recommended)

---

## Week 1: React Fundamentals

### Goals
- Understand what React is and why it's used
- Set up a React development environment
- Create your first React application

### Topics
- Introduction to React and its ecosystem
- Setting up with Create React App or Vite
- JSX syntax and expressions
- Components: functional vs class components
- Props basics

### Project
Create a simple "Hello World" React app with multiple components displaying your profile information.

### Resources
- [React Official Documentation](https://react.dev/)
- [Create React App](https://create-react-app.dev/)
- [Vite](https://vitejs.dev/)

---

## Week 2: Components and Props

### Goals
- Master component composition
- Understand props and data flow
- Learn component best practices

### Topics
- Component hierarchy and composition
- Passing props between components
- Default props and prop types
- Children props
- Component reusability patterns

### Project
Build a reusable card component library with various card types (profile card, product card, article card).

### Resources
- [Thinking in React](https://react.dev/learn/thinking-in-react)
- [Passing Props to a Component](https://react.dev/learn/passing-props-to-a-component)

---

## Week 3: State and Event Handling

### Goals
- Understand React state management
- Handle user events effectively
- Control form inputs

### Topics
- useState hook introduction
- State updates and re-rendering
- Event handling (onClick, onChange, onSubmit)
- Controlled vs uncontrolled components
- Form handling basics

### Project
Create an interactive todo list application with add, complete, and delete functionality.

### Resources
- [State: A Component's Memory](https://react.dev/learn/state-a-components-memory)
- [Responding to Events](https://react.dev/learn/responding-to-events)

---

## Week 4: React Hooks Deep Dive

### Goals
- Master essential React hooks
- Understand when and how to use each hook
- Learn hook rules and best practices

### Topics
- useEffect for side effects
- useRef for DOM references and values
- useMemo and useCallback for optimization
- Rules of hooks
- Custom hooks introduction

### Project
Build a weather app that fetches data from an API and displays current conditions with loading states.

### Resources
- [Using Effects](https://react.dev/learn/synchronizing-with-effects)
- [Hooks Reference](https://react.dev/reference/react)

---

## Week 5: Styling in React

### Goals
- Explore different styling approaches
- Implement responsive designs
- Choose appropriate styling solutions

### Topics
- CSS Modules
- Styled-components
- Tailwind CSS integration
- CSS-in-JS patterns
- Responsive design techniques

- Create multi-page applications
- Handle navigation and URL parameters

- Route parameters and query strings
- Nested routes and layouts

### Project

## Week 7: State Management

### Goals
- Understand different state management approaches
- Implement global state solutions
- Choose appropriate patterns for your apps

### Topics

- Introduction to Redux Toolkit

- When to use local vs global state

- Deploy your applications

- **Next.js** - Server-side rendering and full-stack React

- **GraphQL with Apollo** - Modern API layer
- **Accessibility (a11y)** - Build inclusive applications
